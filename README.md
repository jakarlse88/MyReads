# MyReads

A bookshelf app that allows selection and categorisation of books, made as part of Udacity's Front-End Developer Nanodegree. Built from the ground up using create-react-app and a provided API server/client library. 

## Usage

* Clone or download this repo
* Navigate to the repo directory using a terminal
* Run `npm install`
* Start the webserver using either `npm start` 
* Using your favourite browser, navigate to `localhost:3000`
